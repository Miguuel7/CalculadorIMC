# Calculadora de IMC ⚖️

## Descrição

Este projeto consiste em uma **calculadora de IMC (Índice de Massa Corporal)** desenvolvida utilizando **HTML, CSS e JavaScript**.

O objetivo é permitir que o usuário insira seu **peso** e **altura**, e ao clicar no botão **Calcular**, o sistema calcula automaticamente o valor do IMC e exibe a classificação correspondente com base na tabela da Organização Mundial da Saúde (OMS).

O projeto foi criado para demonstrar conceitos básicos de **interatividade em páginas web** e manipulação de elementos da interface utilizando JavaScript.

---

## Funcionalidades

* Inserção de **peso (kg)** e **altura (m)** pelo usuário
* Botão para realizar o cálculo do IMC
* Exibição do valor calculado
* Classificação do resultado conforme a tabela da OMS

---

## Tecnologias utilizadas

* **HTML** – estrutura da página
* **CSS** – estilização e organização visual
* **JavaScript** – cálculo do IMC e exibição do resultado

---

## Conceitos aplicados

Durante o desenvolvimento foram utilizados alguns conceitos importantes de JavaScript:

* `document.getElementById()`
* `addEventListener()`
* `parseFloat()`
* `Math.pow()`
* estruturas condicionais (`if`, `else if`, `else`)

---

## Fórmula utilizada

IMC = peso / altura²

---

## Objetivo do projeto

Este projeto foi desenvolvido com fins **educacionais**, com o objetivo de praticar o desenvolvimento de aplicações web simples e compreender como **HTML, CSS e JavaScript** podem ser utilizados juntos para criar páginas interativas.
